# DrawProcedureNow 接口

可以使用 DrawProcedureNow 接口来绘制粒子效果 (但是只有 ES4.5 以上才可能支持), 但是这个其实完全可以用 instance 来代替，主要的作用是: 不需要输入顶点,顶点数量是procedure中指定的,其他的数据都是从SSBO中获取的 ，而牵扯到 SSBO , 那就估计就要 ES5.0 以后才可以完全支持
DrawProcedure 触发的时候，可以不用每帧提交
