# 第一个动效

注意：这个仅由 css3 控制的动效仅适应如上图简单的条件下，复杂的条件下 (可以用 js 实现),css (暂时不会) 思路是：将当前的 container 作为一个 relative 容器，其子容器均为 abslute, 根据 top,left,bottom.right 来调整大小
设置一个子容器使用 sv-copy-rotate, 只填充颜色.
设置其他的 z-index 均大于该层.
然后设置旋转动画
