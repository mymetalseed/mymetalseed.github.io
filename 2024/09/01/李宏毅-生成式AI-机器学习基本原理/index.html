# 根据函数来分类

回归 (Regression): 输入经过函数 (抽象的函数) 输出一个值
分类 (Classification): 函数的输出是一个类别

但是机器学习还有一个很困难的问题，就是 structured learning

  • 生成有结构的物件：影相、文句
  • 又叫生成式学习

深度学习中类神经网络的结构：如 CNN、RNN、Transformer 等等，就是待选函式结构的集合。

# 生成式学习的两种策略

各个击破、一次到位

各个击破：如一个文字一个文字生成
一次到位：一次输出就是一段文字或者图像

# 生成复杂物体

对于对应的场景，可以将物体拆分成单位，一些常见的单位如下:
文本: token
影像：像素
语音：采样点
