# 经济学研究什么

  • 重商主义：货币是财富的唯一形态，对外贸易是财富的唯一源泉
  • 重农学派：自然秩序为最高信条

1776.3.9 亚当・斯密发表《国民财富的性质和原因的研究（国富论）》后，经济学正式成为一门学科

  • 分工的好处
    • 分工可以减少工作之间往返的成本
    • 分工促进熟能生巧
    • 分工使机器替代成为肯呢个

# “政治经济学” 与 “经济学”

政治经济学 -> 经济学

马克思、恩格斯批评政治经济学的研究范围太狭隘，认为还需要研究经济活动以外的社会政治活动和关系
慢慢的 政治经济学 逐渐被 经济学 替代

# 经济与经济学

  • 经济 (economy): 如何以最小的代价，取得最大的效果，即在各种条件的制约下，选取代价最小而收效最大的那种选择
    • 经济人
    • 理性人
  • 经济学（economics）: 研究如何将稀缺资源用于最有效的用途，简称 "资源配置问题"

# 秩序

  • 亚当斯密时代，大多数人都相信，只有统治者的精心筹划才能使一个社会免于混乱与贫困

经济学的思维方式，所有的社会现象均源于个体的行为以及群体的合作:

  • 个体的行为
  • 群体的合作
  • 前述两种活动的结果
